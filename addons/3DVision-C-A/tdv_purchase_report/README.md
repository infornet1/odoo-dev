<h1>tdv_purchase_report</h1>

## Models
**PurchaseReportLine:** Representa una línea individual dentro de un reporte de compras, proporcionando una estructura detallada para almacenar y calcular la información relevante de cada factura de compra

**PurchaseReport:** Consolida las facturas de compras dentro de un período definido, proporcionando una visión detallada de los montos y los impuestos asociados a sus compras.

## Reports
**PurchaseReportXlsx:** Genera un informe en formato XLSX (Excel) basado en los datos del reporte de compras (tdv.purchase.report). Este informe está diseñado para presentar de manera estructurada y detallada la información de las facturas de compra, sus montos e impuestos asociados.
